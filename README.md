# Portfolio V2

Professional Web Portfolio developped With **Passion** by _Boll_.

Tech Stack : 
- [Vite](https://vitejs.dev/)
- [React.js](https://reactjs.org/)
- [Three.js](https://threejs.org/)
- [GSAP](https://greensock.com/)
- [Tailwind CSS](https://tailwindcss.com/) (4.1)
- Icons using [Iconoir](https://iconoir.com/)
- Card tilt using [vanilla-tilt](https://micku7zu.github.io/vanilla-tilt.js/)
- Contact form using [EmailJS](https://www.emailjs.com/)
- Avatar 3D model using [Ready Player Me](https://readyplayer.me/)
- 3D model avatar animations using [Mixamo](https://www.mixamo.com/)
- 2D Physics simulation using [matter.js](https://brm.io/matter-js/)


Utils : R3F-Perf, Leva, R3F, R3F-Drei
